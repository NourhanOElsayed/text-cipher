# TextCipher

A cross-platform desktop application built to visualize classical cryptography algorithms. TextCipher provides real-time, step-by-step visualizations of mathematical encryption and decryption processes, designed with a clean, dark-mode user interface.

## Features

* **Interactive Visualizations:** Watch the encryption process step-by-step as plaintext transforms into ciphertext.
* **Multiple Cipher Support:**
    * Caesar Cipher (Shift)
    * Vigenère Cipher *(Coming Soon)*
    * Monoalphabetic Cipher *(Coming Soon)*
* **Real-time Mathematical Breakdown:** Displays the underlying modulo arithmetic for each character transformation (e.g., `H = (7 + 10) mod 26 = 17 -> R`).

## Tech Stack

* **Framework:** [Flutter](https://flutter.dev/)
* **Language:** Dart
* **Target Platforms:** Windows
